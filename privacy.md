# Privacy Policy

**App name:** Automation Lab OS (Dev)  
**Last updated:** 2026-01-30

## Summary
Automation Lab OS is an internal automation tool used by the operator only.

## Data we collect

## Data retention
- Access tokens and related metadata may be stored securely for automation purposes.
- Data is retained only as long as needed for internal operations and can be deleted upon request.
